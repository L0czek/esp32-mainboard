# TMP107 Temperature Sensor Chain — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add TMP107 daisy-chain temperature sensor support — auto-discover sensors at boot, poll at 10Hz, publish raw readings to MQTT.

**Architecture:** Shared driver in `src/tmp107.rs` (reusable library module) owns UART + direction pin, handles the TMP107 SMAART wire protocol. Binary-specific collection task in `src/bin/test_stand_controller/temperature_collection.rs` configures hardware, initializes the driver, and runs the 10Hz publish loop using the existing MQTT temperature packet infrastructure.

**Tech Stack:** esp-hal 1.0 (UART, GPIO), Embassy (async runtime, Ticker, with_timeout), existing MQTT queue (`publish_temperature_sensor`/`TempPacket`)

**Testing:** This is `no_std` embedded firmware. Verification is `cargo check --bin test_stand_controller` + `cargo clippy` at each step, then on-device smoke testing after final step. See AGENTS.md for env vars needed.

**Reference:** TMP107 datasheet at `docs/tmp107.pdf`. Constants marked `// VERIFY: datasheet` should be cross-checked against it.

```rust
use defmt::info;
use embassy_time::{with_timeout, Duration, Timer};
use esp_hal::gpio::{Level, Output};
use esp_hal::uart::{UartRx, UartTx};
use esp_hal::Async;

// -- Protocol constants (VERIFY: datasheet docs/tmp107.pdf) --

/// Maximum sensors in a TMP107 daisy chain (5-bit address space).
pub const MAX_SENSORS: usize = 32;

/// Sent before every command so sensors can auto-detect baud rate.
const CALIBRATION_BYTE: u8 = 0x55;

/// Address Initialize command byte.
const ADDR_INIT_COMMAND: u8 = 0x95;

/// Temperature register address.
const TEMP_REGISTER: u8 = 0x00;

/// Timeout for a single sensor response (individual read).
const READ_TIMEOUT_MS: u64 = 10;

/// Timeout waiting for next address-init response before giving up.
const ADDR_DISCOVER_TIMEOUT_MS: u64 = 50;

/// Time for 3 TX bytes to physically leave the wire at 115200 baud
/// (3 * 10 bits / 115200 ≈ 260 µs). Rounded up for safety.
const TX_DRAIN_US: u64 = 500;

// -- Error type --

#[derive(Debug, Clone, Copy, defmt::Format)]
pub enum Tmp107Error {
    UartWrite,
    UartRead,
    Timeout,
    NoSensorsFound,
}

// -- Driver struct --

pub struct Tmp107 {
    tx: UartTx<'static, Async>,
    rx: UartRx<'static, Async>,
    dir: Output<'static>,
    sensor_count: u8,
}
```

**Step 1: Add helper functions for byte construction**

```rust
impl Tmp107 {
    /// Build command byte: [G/nI][R/nW][0][A4..A0]
    fn command_byte(global: bool, read: bool, address: u8) -> u8 {
        let mut byte: u8 = address & 0x1F;
        if global {
            byte |= 0x80;
        }
        if read {
            byte |= 0x40;
        }
        byte
    }

    /// Build register pointer byte: [P3..P0][0101]
    fn register_pointer(register: u8) -> u8 {
        ((register & 0x0F) << 4) | 0x05
    }
}
```

**Step 2: Add half-duplex TX helper**

Sends calibration + command + register pointer, then switches to RX mode.

```rust
    async fn send_command(
        &mut self,
        global: bool,
        read: bool,
        address: u8,
        register: u8,
    ) -> Result<(), Tmp107Error> {
        let cmd = Self::command_byte(global, read, address);
        let ptr = Self::register_pointer(register);
        let bytes = [CALIBRATION_BYTE, cmd, ptr];

        self.dir.set_high();
        self.tx
            .write_async(&bytes)
            .await
            .map_err(|_| Tmp107Error::UartWrite)?;
        // Wait for bytes to physically leave the wire before switching
        Timer::after_micros(TX_DRAIN_US).await;
        self.dir.set_low();

        Ok(())
    }
```

**Step 3: Add read-exact helper with timeout**

Reads exactly `buf.len()` bytes, retrying partial reads.

```rust
    async fn read_exact(&mut self, buf: &mut [u8]) -> Result<(), Tmp107Error> {
        let mut offset = 0;
        while offset < buf.len() {
            let n = with_timeout(
                Duration::from_millis(READ_TIMEOUT_MS),
                self.rx.read_async(&mut buf[offset..]),
            )
            .await
            .map_err(|_| Tmp107Error::Timeout)?
            .map_err(|_| Tmp107Error::UartRead)?;

            if n == 0 {
                return Err(Tmp107Error::UartRead);
            }
            offset += n;
        }
        Ok(())
    }
```

**Step 4: Implement the four protocol primitives**

```rust
    async fn individual_read(
        &mut self,
        address: u8,
        register: u8,
    ) -> Result<u16, Tmp107Error> {
        self.send_command(false, true, address, register).await?;
        let mut buf = [0u8; 2];
        self.read_exact(&mut buf).await?;
        Ok(u16::from_le_bytes(buf))
    }

    async fn global_read(
        &mut self,
        max_address: u8,
        register: u8,
        out: &mut [u16],
    ) -> Result<usize, Tmp107Error> {
        let count = (max_address as usize).min(out.len());
        self.send_command(true, true, max_address, register)
            .await?;
        let mut buf = [0u8; 2];
        for slot in out.iter_mut().take(count) {
            self.read_exact(&mut buf).await?;
            *slot = u16::from_le_bytes(buf);
        }
        Ok(count)
    }

    async fn individual_write(
        &mut self,
        address: u8,
        register: u8,
        value: u16,
    ) -> Result<(), Tmp107Error> {
        self.send_command(false, false, address, register).await?;
        // After send_command, dir is already LOW. Switch back to TX.
        self.dir.set_high();
        let bytes = value.to_le_bytes();
        self.tx
            .write_async(&bytes)
            .await
            .map_err(|_| Tmp107Error::UartWrite)?;
        Timer::after_micros(TX_DRAIN_US).await;
        self.dir.set_low();
        Ok(())
    }

    async fn global_write(
        &mut self,
        max_address: u8,
        register: u8,
        value: u16,
    ) -> Result<(), Tmp107Error> {
        self.send_command(true, false, max_address, register).await?;
        self.dir.set_high();
        let bytes = value.to_le_bytes();
        self.tx
            .write_async(&bytes)
            .await
            .map_err(|_| Tmp107Error::UartWrite)?;
        Timer::after_micros(TX_DRAIN_US).await;
        self.dir.set_low();
        Ok(())
    }
```

## Task 3: Implement init and public API

**Files:**
- Modify: `src/tmp107.rs`

**Step 1: Implement `Tmp107::init`**

This is the combined constructor + address discovery.

```rust
impl Tmp107 {
    /// Create driver, run Address Initialize, return configured driver
    /// with discovered sensor count.
    pub async fn init(
        tx: UartTx<'static, Async>,
        rx: UartRx<'static, Async>,
        dir: Output<'static>,
    ) -> Result<Self, Tmp107Error> {
        let mut driver = Self {
            tx,
            rx,
            dir,
            sensor_count: 0,
        };

        driver.discover_sensors().await?;
        Ok(driver)
    }

    async fn discover_sensors(&mut self) -> Result<(), Tmp107Error> {
        // Send: calibration + address init command + starting address (1)
        let bytes = [CALIBRATION_BYTE, ADDR_INIT_COMMAND, 0x01];

        self.dir.set_high();
        self.tx
            .write_async(&bytes)
            .await
            .map_err(|_| Tmp107Error::UartWrite)?;
        Timer::after_micros(TX_DRAIN_US).await;
        self.dir.set_low();

        // Read address responses until timeout (each sensor responds
        // with 1 byte at ~7 ms intervals)
        let mut count: u8 = 0;
        let mut response = [0u8; 1];
        loop {
            match with_timeout(
                Duration::from_millis(ADDR_DISCOVER_TIMEOUT_MS),
                self.rx.read_async(&mut response),
            )
            .await
            {
                Ok(Ok(1)) => {
                    count += 1;
                    info!(
                        "TMP107 sensor {} discovered (byte: {:#04x})",
                        count, response[0]
                    );
                    if count as usize >= MAX_SENSORS {
                        break;
                    }
                }
                _ => break,
            }
        }

        if count == 0 {
            return Err(Tmp107Error::NoSensorsFound);
        }

        self.sensor_count = count;
        info!("TMP107 init complete: {} sensors", count);
        Ok(())
    }
}
```

**Step 2: Implement public read API**

```rust
    pub fn sensor_count(&self) -> u8 {

    /// Read temperature from a single sensor by address (1-based).
    pub async fn read_temperature(
        &mut self,
        address: u8,
    ) -> Result<u16, Tmp107Error> {
        self.individual_read(address, TEMP_REGISTER).await

    /// Read temperatures from all discovered sensors via global read.
    /// Returns the number of readings written to `out`.
    /// `out` must be at least `sensor_count()` elements long.
    pub async fn read_all_temperatures(
        &mut self,
        out: &mut [u16],
    ) -> Result<usize, Tmp107Error> {
        self.global_read(self.sensor_count, TEMP_REGISTER, out)
            .await
    }
```

```rust
use defmt::{error, info, warn};
use embassy_time::{Duration, Instant, Ticker};
use esp_hal::gpio::{Level, Output, OutputConfig};
use esp_hal::uart::Uart;
use esp_hal::peripherals::UART0;
use esp_hal::time::Rate;

use crate::config::TEMP_COLLECTION_INTERVAL_MS;
use crate::mqtt::publish_temperature_sensor;
use crate::mqtt::sensors::temp::TempPacket;
use mainboard::board::{D0Pin, U0RxPin, U0TxPin};
use mainboard::tmp107::{Tmp107, MAX_SENSORS};

pub struct TemperatureCollectionIo {
    pub uart: UART0<'static>,
    pub tx_pin: U0TxPin,
    pub rx_pin: U0RxPin,
    pub dir_pin: D0Pin,
}

#[embassy_executor::task]
pub async fn temperature_collection_task(io: TemperatureCollectionIo) {
    let uart = Uart::new(
        io.uart,
        esp_hal::uart::Config::default()
            .with_baudrate(Rate::from_hz(115200)),
    )
    .expect("UART0 init failed")
    .with_tx(io.tx_pin)
    .with_rx(io.rx_pin)
    .into_async();

    let (rx, tx) = uart.split();

    let dir = Output::new(io.dir_pin, Level::Low, OutputConfig::default());

    let mut driver = match Tmp107::init(tx, rx, dir).await {
        Ok(d) => d,
        Err(e) => {
            error!("TMP107 init failed: {:?}", e);
            return;
        }
    };

    info!(
        "Temperature collection: {} sensors, {}ms interval",
        driver.sensor_count(),
        TEMP_COLLECTION_INTERVAL_MS,
    );

    let mut ticker = Ticker::every(Duration::from_millis(
        TEMP_COLLECTION_INTERVAL_MS,
    ));
    let mut buf = [0u16; MAX_SENSORS];

    loop {
        ticker.next().await;

        let timestamp_ms = Instant::now().as_millis() as u32;

        let count = match driver.read_all_temperatures(&mut buf).await {
            Ok(n) => n,
            Err(e) => {
                warn!("TMP107 read failed: {:?}", e);
                continue;
            }
        };

        for i in 0..count {
            let sensor_id = (i + 1) as u8;
            let packet = match TempPacket::from_slice(
                sensor_id,
                timestamp_ms,
                &[buf[i]],
            ) {
                Ok(p) => p,
                Err(e) => {
                    warn!(
                        "TMP107 packet error sensor {}: {:?}",
                        sensor_id, e,
                    );
                    continue;
                }
            };

            if publish_temperature_sensor(packet).is_err() {
                warn!("Dropping temp packet: queue full");
            }
        }
    }
}
```

## Task 7: On-device verification

**Not automatable — manual checklist for the developer:**

1. Flash: `WIFI_SSID=... WIFI_PASSWORD=... MQTT_HOST=... cargo espflash --release --bin test_stand_controller /dev/ttyUSB0`
2. Check RTT log for:
   - `TMP107 sensor 1 discovered` (one per sensor in chain)
   - `TMP107 init complete: N sensors`
   - `Temperature collection: N sensors, 100ms interval`
3. Subscribe to MQTT: `mosquitto_sub -h <host> -t 'sensor/temp/#' -v`
4. Confirm messages arrive at ~10Hz per sensor
5. Verify temperature values are plausible (room temp ≈ 1400–1700 raw for 22–27°C)

If init fails with `NoSensorsFound`:
- Check wiring: U0Tx→bus driver→sensor chain, D0→bus driver enable
- Verify bus driver direction: D0 HIGH = TX, D0 LOW = RX
- Try lower baud rate (change `Rate::from_hz(115200)` to `Rate::from_hz(9600)`)
